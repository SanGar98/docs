## Development Environment

Setting up a proper development environment is crucial for efficient documentation development.

### Prerequisites

Before you begin, ensure you have the following installed:

<CardGroup cols={2}>
  <Card title="Node.js" icon="node-js">
    Version 14.0 or higher required
  </Card>
  <Card title="Git" icon="github">
    For version control and collaboration
  </Card>
</CardGroup>

### Installation Steps

<Steps>
  <Step title="Install Mintlify CLI">
    Install the Mintlify CLI globally using npm:
    ```bash
    npm install -g mintlify
    ```
  </Step>
  <Step title="Clone Repository">
    Clone your documentation repository:
    ```bash
    git clone https://github.com/SanGar98/docs.git
    cd docs
    ```
  </Step>
  <Step title="Start Development Server">
    Run the development server:
    ```bash
    mintlify dev
    ```
    Your docs will be available at `http://localhost:3000`
  </Step>
</Steps>

## Development Workflow

1. **Create a new branch** for your changes
2. **Make your edits** using your preferred editor
3. **Preview locally** using `mintlify dev`
4. **Commit your changes** with descriptive messages
5. **Push to GitHub** to trigger automatic deployment

<Tip>
  Use the Mintlify VS Code extension for better syntax highlighting and autocomplete!
</Tip>
