# Control de Cambios Vortexbird

Control de Cambios es una aplicación empresarial desarrollada por Vortexbird para gestionar y rastrear cambios en proyectos de software de manera eficiente y organizada.

## Overview

Este sistema permite a las empresas y equipos de desarrollo:

- **Gestionar Registros de Control**: Crear, actualizar y rastrear cambios en proyectos
- **Administrar Bolsas de Horas**: Controlar la asignación y consumo de horas por proyecto
- **Gestión de Desarrolladores**: Administrar el equipo de desarrollo y sus asignaciones
- **Control de Proyectos**: Organizar y monitorear múltiples proyectos y empresas
- **Generación de Reportes**: Crear reportes en PDF con información detallada de cambios
- **Historial de Cambios**: Mantener un registro completo de todas las modificaciones realizadas

## Technology Stack

- **Backend Framework**: Spring Boot 2.7.3
- **Language**: Java 17
- **Database**: PostgreSQL
- **Security**: Spring Security + JWT
- **API Documentation**: SpringDoc OpenAPI
- **Object Mapping**: MapStruct
- **Reports**: JasperReports 6.20.0
- **Inter-service Communication**: OpenFeign
