---
title: "GET Request"
api: "GET https://api.example.com/v1/users/{id}"
description: "Retrieve a user by their ID"
---

## Overview

This endpoint retrieves detailed information about a specific user.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the user
</ParamField>

## Query Parameters

<ParamField query="include" type="string">
  Comma-separated list of related resources to include (e.g., "profile,settings")
</ParamField>

<ParamField query="fields" type="string">
  Comma-separated list of fields to return
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  The user object
  
  <Expandable title="User Object">
    <ResponseField name="id" type="string">
      Unique identifier for the user
    </ResponseField>
    
    <ResponseField name="email" type="string">
      User's email address
    </ResponseField>
    
    <ResponseField name="name" type="string">
      User's full name
    </ResponseField>
    
    <ResponseField name="created_at" type="timestamp">
      When the user was created
    </ResponseField>
    
    <ResponseField name="updated_at" type="timestamp">
      When the user was last updated
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl https://api.example.com/v1/users/123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/v1/users/123', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/v1/users/123',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

data = response.json()
print(data)
```

</CodeGroup>
