## Images

### Basic Image

Add images using standard markdown syntax:

```markdown
![Alt text](/images/example.png)
```

### Responsive Images

Use Mintlify's image component for responsive images:

```markdown
<img
  src="/images/example.png"
  alt="Example"
  width="600"
/>
```

### Dark Mode Images

Show different images for light and dark modes:

```markdown
<img
  className="block dark:hidden"
  src="/images/hero-light.svg"
  alt="Hero Light"
/>
<img
  className="hidden dark:block"
  src="/images/hero-dark.svg"
  alt="Hero Dark"
/>
```

## Image Organization

Store your images in the `/images` directory:

```
images/
├── logo/
│   ├── dark.svg
│   └── light.svg
├── screenshots/
│   └── dashboard.png
└── diagrams/
    └── architecture.svg
```

## Videos

### Embedded Videos

Embed videos from popular platforms:

```markdown
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="YouTube video player"
  frameborder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen
></iframe>
```

### Video Callout

Use the Frame component for better video integration:

<Frame>
  <img src="https://via.placeholder.com/800x450" alt="Video placeholder" />
</Frame>

## Diagrams

### Mermaid Diagrams

Create diagrams using Mermaid syntax:

```mermaid
graph TD
    A[Start] --> B{Is it working?}
    B -->|Yes| C[Great!]
    B -->|No| D[Debug]
    D --> B
```

## Icons

Use Font Awesome icons in your documentation:

```markdown
<Icon icon="check" /> Success
<Icon icon="warning" /> Warning
<Icon icon="info" /> Information
```
